## Install Instructions

* Open a Terminal
* `cd ~/Dropbox`
* `git clone https://github.com/shawnrice/transcript-generator.git`
* `mv transcript-generator transcripts`
* `cd transcripts`
* `npm install`
* `node index.js`

## Build from Finder
Just double-click `build.sh`

## Personalize
Open `config.js` and change the values.
